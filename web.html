Web design portfolio

From 2005 to 2017, I worked at Brown University primarily as a Web graphic designer. I was also heavily involved with building web architecture, content strategy, UX design, user testing, and project management.

Starting in 2017, I have been directing my efforts toward front-end development using WordPress. I create and modify WordPress themes and subthemes using tools such as Sass, Bootstrap, and Git.
